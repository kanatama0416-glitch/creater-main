# Card Image Slots

このフォルダにはカード一覧の仮画像を10枚用意しています。

- card-01.svg
- card-02.svg
- card-03.svg
- card-04.svg
- card-05.svg
- card-06.svg
- card-07.svg
- card-08.svg
- card-09.svg
- card-10.svg

使い方:
1. `card-xx` の名前を維持して画像を置く
2. 拡張子は `.png` `.jpg` `.jpeg` `.webp` `.svg` のいずれかでOK
3. そのまま一覧表示に反映されます

別名の画像を使う場合は `src/lib/localCardSlots.ts` の `image_url` を変更してください。
